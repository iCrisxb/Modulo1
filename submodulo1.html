¿Qué es el diseño de software?

El diseño de software es como crear un plan detallado antes de construir algo. Es el proceso que los programadores usan para organizar sus ideas y reducir errores al construir soluciones informáticas. ¡Es como planear antes de actuar!

JavaScript 

  function mostrarResumen() {
    const tipo = document.getElementById('tipo-pizza').value;
    const tamano = document.getElementById('tamano-pizza').value;
    const extras = Array.from(document.querySelectorAll('input[type="checkbox"]:checked'))
                        .map(input => input.value);
  
    const precio = calcularPrecio(tamano, extras);
  
    document.getElementById('detalle-pedido').innerText = 
      `Pizza: ${tipo}, Tamaño: ${tamano}, Extras: ${extras.join(', ') || 'Ninguno'}`;
    document.getElementById('precio-pedido').innerText = `Precio total: $${precio} MXN`;
  
    document.getElementById('resumen-pedido').classList.remove('oculto');
  }
  
  function calcularPrecio(tamano, extras) {
    let base = tamano === "Grande" ? 200 : 150;
    return base + extras.length * 20;
  }

Explicación:

Este código permite mostrar un resumen del pedido de pizza, calculando el precio basado en el tamaño y los extras seleccionados. También muestra los detalles del pedido en la página.

¿Qué es un diagrama de flujo?

Un diagrama de flujo es una representación gráfica que utiliza símbolos para mostrar la secuencia de pasos en un proceso. Ayuda a comprender, analizar y comunicar cómo funcionan los procesos al dividirlos en pasos simples y claros. Es una herramienta ideal para planificar tareas, resolver problemas y mejorar la eficiencia en proyectos.

×

Símbolos de los diagramas de flujo:

Inicio / Fin

Indica el inicio o el fin de un proceso o diagrama de flujo.

Linea de Flujo

Indica la dirección del flujo del diagrama, conectando los distintos elementos.

Entrada / Salida

Indica el proceso de entrada o salida de datos externos.

Proceso

Representa un proceso o acción, como una operación o tarea a realizar.

Decisión

Muestra un paso que decide el próximo paso en un proceso.
